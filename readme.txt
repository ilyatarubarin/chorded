Chorded

Chorded — Android-приложение для изучения гитарных аккордов и песен с возмождностью подбора песни по известным пользователю аккордам.
Приложение предоставляет каталог аккордов, песни с аккордами и систему отслеживания прогресса пользователя.

Возможности

	Авторизация пользователя и гостевой режим

	Каталог аккордов с визуальными схемами

	Каталог песен с аккордами

	Кликабельные аккорды внутри песни

	Воспроизведение аудио

	Отслеживание прогресса обучения

	Облачное хранение данных с использованием Firebase

Инструкция по запуску
Требования

	Android Studio (рекомендуется последняя стабильная версия)

	Android SDK

	Эмулятор Android или физическое устройство

	Аккаунт Firebase

Установка и настройка

	Клонировать репозиторий:

	git clone https://github.com/your-username/chorded.git


	Открыть проект в Android Studio:

	File → Open
	
	выбрать папку проекта

	Создать проект в Firebase:

	включить Firebase Authentication

	включить Firestore Database

	включить Firebase Storage

	Добавить конфигурацию Firebase:

	скачать файл google-services.json

	поместить его в каталог app/

	Выполнить синхронизацию Gradle.

	Запуск приложения
	
	Выбрать эмулятор или подключённое устройство.

	Нажать кнопку Run в Android Studio.

	Приложение будет запущено на выбранном устройстве.
